Bookshelf API (Activity 3)

Backend built with NestJS + TypeORM + SQLite.

Run (from this folder):

1. Install dependencies

```bash
npm install
```

2. Run in development mode

```bash
npm run start:dev
```

The server will run on http://localhost:3000 and Swagger UI is at http://localhost:3000/api

Notes: This project uses SQLite file `db.sqlite` created automatically by TypeORM.
